# ACSD.css – A Clean Semantic Drop-in

A tiny (~14 KB gzipped), single-file, classless-first CSS framework built for:
- Marketing pages
- Documentation & blogs
- Internal tools

## Features
- Pure semantic HTML looks great out of the box
- Full design token system (colors, spacing, typography, radius…)
- Light + Dark theme via `data-theme="dark"` or `.theme-dark`
- Accessible focus rings & form styling
- Reusable components: Button, Card, Alert, Badge + optional Navbar & Tabs
- Small set of token-driven utility classes
- Layered architecture with `@layer`

## Usage
```html
<link rel="stylesheet" href="https://your-cdn/acsd.css">
